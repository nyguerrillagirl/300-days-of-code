This folder contains p5js projects. 

All the projects used the following:
1. Visual Studio Code
2. Extensions:
	* Live Server by Ritwick Dey
	* p5.vscode

To start a folder/project for a specific p5js program:
1. Open the Command Palette (command-shift-p on Mac or ctrl-shift-p on windows_
2. Enter: Create p5.js Project
3. Select an new empty folder to put your project in
4. Edit the file sketch.js with your p5.js code
5. Run the Live Server "Go Live" to open html page in your browser to see your handywork

Any online tutorials I used the notes are under the project (easier to associate one with the other).
